# react-d3-basic

react-d3 basic charts, charts that we support:

- Line Chart
- Multiple Line Chart
- Scatter Plot
- Area Chart
- Area Stack Chart
- Bar Chart
- Bar Group Chart
- Bar Stack Chart
- Pie Chart
- Donut Chart


## Install

```
npm install --save react-d3-basic
```
