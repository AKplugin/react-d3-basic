# xyplot

This is where plots using x, y axis set their initial props, state and some basic functions.

Such as area chart, area stack chart, bar chart, bar group chart, bar stack chart, line chart... etc.

## function

#### mkXScale

making xscale

props:

- xScale
- xRange
- xDomain
- xRangeRoundBands

#### mkYScale

making yScale

props:

- yScale
- yRange
- yDomain
- yRangeRoundBands

#### mkSeries

making series

props:

- data
- chartSeries
- x
- y
- categoricalColors
